{{- if eq .Values.service.type "ClusterIP" }}

Access the RPC proxy from within the cluster:
  http://{{ include "erpc.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local:{{ .Values.service.port }}

To port-forward for local access:
  kubectl port-forward -n {{ .Release.Namespace }} svc/{{ include "erpc.fullname" . }} {{ .Values.service.port }}:{{ .Values.service.port }}

Then send RPC requests to:
  curl http://localhost:{{ .Values.service.port }}/{{ .Values.projectId }}/evm/{{ .Values.l1.evmChainId }} \

    -d '{"jsonrpc":"2.0","method":"eth_blockNumber","params":[],"id":1}'
